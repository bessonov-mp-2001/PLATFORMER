# Разработка игры в жанре "Платформер"

Краткая характеристика: пользователи, используя игровое приложение, будут осуществлять управление шариком, который будет перемещаться по разным уровням карты, решать различные головоломки и собирать монеты для получения очков. 

## Управление:
- WASD или стрелки - перемещение персонажа
- R - рестарт уровня с точки сохранения
- Esc - выход с уровня
